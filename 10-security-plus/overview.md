Security+ (SY0-701) Overview

This folder contains study notes, domain breakdowns, labs, and summaries for the CompTIA Security+ certification.
Security+ establishes foundational knowledge needed for cloud security, IAM, Zero Trust, incident response, and GRC roles.

Security+ Domain Weights:
1. General Security Concepts (28 percent)
2. Threats, Vulnerabilities, and Mitigations (30 percent)
3. Security Architecture (18 percent)
4. Security Operations (14 percent)
5. Security Program Management and Oversight (10 percent)

Purpose of This Folder:
- Reinforce foundational cyber security concepts
- Support preparation for cloud security, IAM, and SC-300
- Provide interview-ready security explanations
- Map Security+ concepts to cloud platforms (AWS and Azure)
