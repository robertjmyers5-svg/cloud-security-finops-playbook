# Cloud Controls Framework

Defines mandatory controls across all cloud environments.

---

# Identity Controls
- MFA  
- Least privilege  
- No standing admin  
- Access reviews  

# Network Controls
- Private endpoints  
- No public exposure  
- Encryption in transit  

# Data Controls
- Encryption at rest  
- Classification  
- Retention  

# Logging Controls
- Centralized logging  
- Immutable audit logs  
- SIEM integration  

# Compliance Controls
- CIS  
- NIST  
- ISO  
- SOC2
