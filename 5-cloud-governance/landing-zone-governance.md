# Landing Zone Governance (AWS + Azure)

Landing Zones are the foundational blueprint for all cloud environments.

---

# Core Landing Zone Components

## 1. Multi-Account / Multi-Subscription Structure
- Sandbox  
- Development  
- Staging  
- Production  
- Security  
- Logging  
- Shared Services  

---

## 2. Identity Architecture
- SSO/Identity Center  
- PIM (Azure)  
- Permission boundaries  
- SCPs/Policies  

---

## 3. Network Architecture
- Hub-and-spoke  
- Private endpoints  
- Centralized firewall  
- VPC/Virtual Network segmentation  

---

## 4. Security Foundations
- GuardDuty / Defender  
- Centralized logging  
- Encryption defaults  
- Patch baselines  
- Vulnerability scanning  

---

## 5. Compliance Foundations
- CIS Benchmarks enforced  
- NIST 800-53 controls  
- Policy-as-Code enforcement  

---

# Summary
A Landing Zone sets the structure, guardrails, and governance for everything in the cloud.
