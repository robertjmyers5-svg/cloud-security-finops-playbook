# Power BI — Cloud Cost Dashboard

This dashboard shows:
- Monthly spend
- Spend by service
- Spend by account/subscription
- Spend by environment (dev, test, prod)
- RI/SP savings impact
- Cost anomalies
- Forecast vs Actual
- Waste (idle/unused resources)

## Metrics
- Total cloud spend
- Cost trend 6–12 months
- Cost per application
- Cost per cost center
- Unallocated spend %
- RI/SP coverage %
- RI/SP utilization %
- Savings opportunities $
- Idle resource hours

## Visuals
- Area chart: Spend over time
- Treemap: Spend by resource type
- Table: Underutilized resources
- KPI cards: RI/SP metrics
- Heatmap: Environment cost breakdown
