Governance Comparison

AWS Governance:
- AWS Organizations
- Service Control Policies (SCPs)
- Config Rules
- Control Tower

Azure Governance:
- Azure Policy
- Management Groups
- Blueprints
- CAF Landing Zones

GCP Governance:
- Resource Hierarchy
- Organization Policies
- Cloud Asset Inventory

Key Differences:
- Azure has strongest policy system (Azure Policy)
- AWS has best multi-account model
- GCP has the simplest hierarchy

Multi-Cloud Strategies:
- Use policy-as-code across clouds
- Enforce tagging standards
- Require identity protection across all clouds
- Centralize logging
