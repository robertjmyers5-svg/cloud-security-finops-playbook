Logging and Monitoring Comparison

AWS:
- CloudTrail (audit logs)
- CloudWatch Logs and Metrics
- GuardDuty
- Config

Azure:
- Activity Logs (audit)
- Log Analytics
- Azure Monitor
- Sentinel

GCP:
- Cloud Logging
- Cloud Monitoring
- Cloud Audit Logs

Multi-Cloud Summary:
- All clouds provide audit logs, metrics, and SIEM integration.
- Azure Sentinel is the strongest built-in SIEM.
- AWS GuardDuty is strong for threat detection.
- GCP logs are unified and simple to query.

Logging Priority:
1. Identity logs
2. Network logs
3. Compute logs
4. Storage access logs
