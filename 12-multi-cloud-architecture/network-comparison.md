Network Comparison (AWS vs Azure vs GCP)

AWS Networking:
- VPC, Subnets
- Security Groups
- NACLs
- Route Tables
- VPC Peering
- Transit Gateway

Azure Networking:
- VNet, Subnets
- NSGs and ASGs
- Route Tables (UDRs)
- VNet Peering
- Azure Firewall
- VPN Gateway

GCP Networking:
- VPC (global)
- Subnets (regional)
- Firewall Rules
- Cloud Router
- VPC Peering

Key Differences:
- AWS: Most mature networking model
- Azure: Most enterprise-integrated (identity + network)
- GCP: Simplest and most flexible (global VPC)

Multi-Cloud Takeaways:
- Private networking exists in all clouds
- Identity-aware networking strongest in Azure
- AWS has the most granular control
