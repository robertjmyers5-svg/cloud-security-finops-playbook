# FinOps Dashboards

Dashboards are the backbone of visibility and accountability.

---

# 1. Executive Summary Dashboard
- Total cloud spend
- MoM variance
- Forecast vs actual
- Top cost drivers

# 2. Allocation Dashboard
- Tagging coverage
- Unallocated spend %
- Cost by team/app

# 3. Rightsizing Dashboard
- CPU/Mem utilization
- Idle/underused resources
- Annualized savings

# 4. Commitment Dashboard
- RI/SP coverage %
- Utilization %
- Expirations

# 5. Forecasting Dashboard
- 30/60/90 day forecast
- Seasonal patterns
- Anomaly detection
