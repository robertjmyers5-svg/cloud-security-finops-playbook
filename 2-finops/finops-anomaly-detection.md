# FinOps Anomaly Detection

Protect the business from unexpected spend spikes.

---

# Types of Anomalies
- Short-term spikes  
- Long-term cost creep  
- Runaway workloads  
- Deployment mistakes  
- Data transfer explosions  

---

# Detection Sources
- AWS Anomaly Detection  
- Azure Cost Alerts  
- Power BI  
- Log Analytics  

---

# Response Playbook
1. Identify  
2. Investigate  
3. Escalate  
4. Remediate  
5. Prevent
