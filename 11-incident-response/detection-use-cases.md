Cloud Detection Use Cases

Identity-Based Alerts:
- Login from unusual country
- Impossible travel
- Access key creation anomalies
- Privilege escalation attempts
- MFA disabled

Network Alerts:
- Port scanning
- Suspicious inbound or outbound traffic
- Publicly exposed storage buckets

Compute Alerts:
- Crypto-mining activity
- High CPU usage anomalies
- Unexpected file downloads

Storage Alerts:
- Public ACL changes
- Unauthorized object access
- Large data exfiltration patterns
