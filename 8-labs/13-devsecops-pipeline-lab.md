# Lab 13 — DevSecOps Pipeline (GitHub Actions)

Goal:
Build a secure CI/CD pipeline.

## Steps
1. GitHub Actions workflow:
   - Lint Terraform
   - IaC security scan
   - Dependency scan
   - Secrets detection
   - Apply policy-as-code
   - Deploy infrastructure

2. Use OIDC (no stored secrets).

3. PR → approval → deploy.
