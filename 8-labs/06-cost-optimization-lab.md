# Lab 6 — Cloud Cost Optimization

Goal:
Identify waste and reduce spend.

## AWS Steps
1. Cost Explorer → Top 3 cost drivers.
2. Identify idle EC2 → Stop/rightsizing.
3. Convert GP2 → GP3 storage.
4. Enable S3 Intelligent Tiering.

## Azure Steps
1. Cost Management → Idle VM list.
2. Downsize VM SKU.
3. Change storage tier.
4. Apply budget + alerts.
