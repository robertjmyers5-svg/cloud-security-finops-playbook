# Lab 8 — Enable Full Security Logging

## AWS Steps
1. CloudTrail → All regions.
2. CloudWatch Logs enabled.
3. GuardDuty ON.
4. IAM Access Analyzer.

## Azure Steps
1. Activity Logs → Log Analytics.
2. Defender for Cloud ON.
3. Diagnostic Settings → Workspace.
4. Security alerts enabled.

## Evidence
- Logs flowing to workspace
- GuardDuty/Defender dashboards
