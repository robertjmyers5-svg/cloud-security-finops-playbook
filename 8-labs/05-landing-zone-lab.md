# Lab 5 — Build a Simple Landing Zone

Goal:
Deploy the core parts of a secure landing zone.

## AWS Steps (Control Tower)
1. Enable Control Tower.
2. Create:
   - Security Account
   - Logging Account
   - Shared Services Account
3. Enable:
   - GuardDuty
   - CloudTrail
   - AWS Config

## Azure Steps (CAF Landing Zone)
1. Deploy Management, Security, Connectivity subscriptions.
2. Enable:
   - Defender for Cloud
   - Activity Logs → Log Analytics
3. Configure:
   - NSGs
   - Private DNS zones
