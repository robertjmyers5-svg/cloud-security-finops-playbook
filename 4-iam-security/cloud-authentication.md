# Cloud Authentication

Authentication determines that a user or workload is who they claim to be.

---

# Strong Authentication Standards

## Multi-Factor Authentication (MFA)
Required for:
- Admins  
- Developers  
- Anyone with console access  

Preferred MFA:
- Authenticator Apps  
- FIDO2 Keys  
- Passkeys  

Avoid:
- SMS codes (weak)

---

# Passwordless Authentication
Benefits:
- No password phishing  
- Strong identity assurance  
- Faster sign-in  

Methods:
- FIDO2  
- Temporary Access Pass (Azure)  
- Certificate-based auth  

---

# Risk-Based Authentication Signals
Evaluate:
- Device posture  
- User risk  
- Session anomaly  
- Location  
- Privilege level  
- Application sensitivity  

This is Zero Trust Identity.
