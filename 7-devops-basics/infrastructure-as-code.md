# Infrastructure-as-Code (IaC)

IaC defines cloud resources using code.

Benefits:
- Consistency
- Repeatable environments
- Version control
- Drift detection
- Automated provisioning

Common IaC Tools:
- Terraform (most common)
- AWS CloudFormation
- Azure Bicep
- Pulumi

IaC + Governance:
- Enforce policies
- Standard modules
- Landing zone deployment
