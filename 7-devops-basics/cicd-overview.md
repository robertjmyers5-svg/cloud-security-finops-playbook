# CI/CD Overview (Continuous Integration & Continuous Delivery)

CI/CD automates code testing, validation, and deployment.

## CI (Continuous Integration)
- Code build
- Unit tests
- Security scans
- Static analysis
- Policy checks

## CD (Continuous Delivery)
- Push to environment (dev/test/prod)
- Automated deployments
- Infrastructure changes (IaC)
- Rollback capabilities

Tools:
- GitHub Actions
- Azure DevOps
- GitLab CI
- Jenkins
- AWS CodePipeline

Cloud Governance uses CI/CD for:
- Tag enforcement
- Policy-as-Code checks
- Security scanning
- Deployment approvals
