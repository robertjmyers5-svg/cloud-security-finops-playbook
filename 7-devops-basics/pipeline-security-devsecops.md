# Pipeline Security (DevSecOps)

Security must be embedded into the pipeline.

## DevSecOps Tools
- Static code analysis (SAST)
- Dependency scanning
- Secrets scanning
- IaC security scanning
- Container scanning
- Policy-as-Code enforcement

## Key Controls
- Prevent secrets in repos
- Enforce MFA for Git
- Require signed commits
- Auto-scan pull requests
- Validate Terraform with policies
