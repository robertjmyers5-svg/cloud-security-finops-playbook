# Terraform Basics

Terraform is the leading multi-cloud IaC tool.

## Core Terraform Concepts
- provider
- resource
- variable
- module
- output
- state

## Terraform Workflow
1. Write code
2. Run `terraform init`
3. Run `terraform plan`
4. Run `terraform apply`

## Best Practices
- Use remote state
- Lock state files
- Use modules
- Avoid hardcoded values
- Tag everything
- Run Terraform through CI/CD

Cloud Governance = Terraform + Policy-as-Code + GitOps.
