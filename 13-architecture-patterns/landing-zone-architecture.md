Landing Zone Architecture

Purpose:
- Provide a secure, scalable foundation for cloud environments.

AWS Components:
- Organizations
- Control Tower
- Service Control Policies
- VPC baselines
- Shared services accounts

Azure Components:
- Management Groups
- Azure Policy
- Log Analytics workspace
- Hub-and-spoke VNet design
- CAF landing zone

Key Requirements:
- Identity as the first control plane
- Centralized logging
- Policy enforcement
- Network segmentation
- Automated guardrails
