IAM Automation Playbooks

Role Rotation Automation:
- Rotate IAM role credentials every 90 days.
- Notify user if rotation fails.

Inactive User Automation:
- Detect identities inactive for 30, 60, 90 days.
- Disable login.
- Notify user or security team.

Privileged Access Automation:
- Auto-expire privileged access.
- Auto-review PIM or IAM role usage.
- Remove unused admin roles.

Service Account Automation:
- Enforce non-expiring credentials.
- Disable static keys in IAM.
