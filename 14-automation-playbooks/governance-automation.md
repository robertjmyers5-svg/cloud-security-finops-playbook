Governance Automation Playbooks

Tag Enforcement Automation:
1. Detect untagged resource.
2. Apply default tags.
3. Notify owner or apply deny policy.

Region Restriction Automation:
- Enforce allowed regions using Azure Policy or AWS SCP.
- Auto-stop or delete resources created in unauthorized regions.

Resource Standardization:
- Auto-apply naming convention.
- Auto-move resources to proper management groups or OUs.

Compliance Reporting:
- Generate automated compliance reports daily.
- Send results to security or governance teams.
